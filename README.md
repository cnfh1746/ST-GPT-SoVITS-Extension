这是一个为 SillyTavern 开发的 GPT-SoVITS TTS 播放器扩展，移植自同名油猴脚本。

## 功能特点

*   **多模式识别**：支持角色+对话、角色+情绪+对话、旁白+对话等多种识别模式。
*   **流式播放**：支持长文本流式生成和播放，减少等待时间。
*   **角色管理**：支持为不同角色配置不同的语音模型和语速。
*   **情感支持**：支持情感识别和语音情感合成。
*   **前端适配**：特别适配了美化后的前端界面（如 juus 本体）。
*   **悬浮控制面板**：方便快捷的播放控制和设置入口。

## 安装方法

1.  将本扩展文件夹 `ST-GPT-SoVITS-Extension` 放入 SillyTavern 的 `public/scripts/extensions/` 目录下。
2.  刷新 SillyTavern 页面。
3.  在页面右侧应该会出现一个悬浮的 TTS 控制面板。
4.  如果没有出现，可以在 SillyTavern 的扩展设置（Extensions -> GPT-SoVITS TTS Player）中点击重置按钮。

## 配置

1.  点击悬浮面板上的齿轮图标打开设置。
2.  配置 TTS API 地址（默认为 `http://127.0.0.1:8000`）。
3.  选择 API 版本。
4.  配置默认语音和角色语音。

## 注意事项

*   本扩展依赖于本地运行的 GPT-SoVITS 服务。
*   请确保 SillyTavern 能够访问到 TTS API 地址（如果存在跨域问题，请配置允许跨域或使用反向代理）。
