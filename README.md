# ST-GPT-SoVITS-Extension

SillyTavern 的 GPT-SoVITS TTS 播放器扩展，移植自油猴脚本。


## ✨ 功能

- **多模式识别** - 角色对话、角色情绪对话、旁白对话等 6 种识别模式
- **角色语音管理** - 按角色配置语音、版本、语速
- **角色分组** - 批量管理角色语音设置
- **提示词注入** - 内置 2 个预设，支持自定义
- **流式播放** - 长文本分段生成和播放
- **边缘隐藏** - 悬浮面板可隐藏至屏幕边缘
- **前端美化适配** - 支持 juus 等美化前端
- **日/夜主题切换**

## 📦 安装

1. 将本文件夹放入 `SillyTavern/public/scripts/extensions/third-party/`
2. 刷新 SillyTavern 页面
3. 右侧出现悬浮 TTS 控制面板

## ⚙️ 配置

1. 点击悬浮面板的 ⚙️ 打开设置
2. 配置 TTS API 地址（默认 `http://127.0.0.1:8000`）
3. 选择 API 版本（v2/v3/v4）
4. 测试连接并刷新模型

## 📌 注意

- 需本地运行 GPT-SoVITS 服务
- 跨域问题请配置 CORS 或使用反向代理
